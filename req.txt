Check if Homebrew is installed on Mac.

If not installed, install Homebrew using official command.

Verify installation using:
brew --version


Install Python 3 using Homebrew.

Command:
brew install python

Verify:
python3 --version



Create virtual environment inside project folder.

Commands:

cd Park_project_2.0
python3 -m venv venv

Activate:
source venv/bin/activate

Goal:
Isolate project dependencies


Install all required Python packages for the project.

Command:

pip install ultralytics opencv-python face_recognition numpy pandas fastapi uvicorn requests

If face_recognition fails, install:

brew install cmake
brew install dlib

Then run:
pip install face_recognition

Goal:
Install all dependencies successfully
